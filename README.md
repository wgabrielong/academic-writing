# academic-writing

Personal LaTeX notes and papers. Every push rebuilds only the documents that
changed (via [Tectonic](https://tectonic-typesetting.github.io/)) and
publishes the resulting PDFs to GitHub Pages, so anything below opens
directly in the browser.

[![build pdfs](https://github.com/wgabrielong/academic-writing/actions/workflows/build-pdfs.yml/badge.svg)](https://github.com/wgabrielong/academic-writing/actions/workflows/build-pdfs.yml)
